--------------------------------------------------------
CORE RESOURCES
--------------------------------------------------------

Basic Installation:

* Put core_credits in resources folder
* Start the resource in your cfg file

COMMANDS:

/givecredits (id) (credits)
(ONLY CONSOLE) /givecreditsrcon (identifier) (credits)

/setcredits (id) (credits)
(ONLY CONSOLE) /setcreditsrcon (identifier) (credits)
